# Netflix-Data-Exploration
This project analyzes a dataset of Netflix movies and TV shows using Python and Jupyter/Colab. The notebook explores various aspects of the Netflix catalog, investigating trends, distributions, and relationships within the data.

Key objectives include:
* Identifying popular genres and movies
* Analyzing the distribution of vote averages and popularity scores
* Comparing genres based on user ratings
* Visualizing changes in popularity and voting patterns over time
* Exploring correlations between movie features like popularity, vote count, and language
* Summarizing insights through tables and visualizations (Matplotlib, Seaborn)

Tech Stack:
Python (Pandas, Matplotlib, Seaborn,Numpy)
Jupyter Notebook / Google Colab
